Installation Steps

Download the Setup File

Click the download button above to get RocketMouseAI_1.0.1.1_Setup.exe.

Run the Installer

Double-click the downloaded Setup.exe. If SmartScreen appears, click "More info" then "Run anyway".

Follow the Instructions

Choose the install location and click "Next". Default: C:\Program Files\RocketMouse AI.

Launch

Open from Start Menu: "Mojosoft" → "RocketMouse AI", or use the desktop shortcut.
